# Telecom Customer Analytics Dashboard

This project was developed as the graduation project for a Business Intelligence diploma/bootcamp using Microsoft Power BI. The dashboard analyzes telecom customer data to deliver actionable insights related to revenue, customer behaviour, churn, retention, segmentation, and customer experience.

The report consists of six interactive pages:

* Executive Summary
* Customer Base & Segments
* Revenue & Monetization
* Usage & Behaviour
* Churn & Retention
* CX & Complaints

Key features:

* Interactive KPI dashboards
* Dynamic slicers and cross-filtering
* Geographic analysis using an Egypt governorate map
* Revenue and ARPU analysis
* Churn and retention monitoring
* Customer segmentation and demographic insights
* Usage behaviour analysis by device and service type
* Customer complaints and NPS analysis

My contribution focused on the Revenue & Monetization and Usage & Behaviour pages, where I worked on KPI development, interactive visualizations, DAX measures, and business insight reporting.

Tools & Technologies:

* Microsoft Power BI
* DAX
* Power Query
